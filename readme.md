## 📖 Descripción del proyecto

El proyecto consiste en el desarrollo de una **Mini CNC (Control Numérico Computarizado)**, es decir, una máquina capaz de automatizar movimientos en distintos ejes mediante comandos programados por computadora.

La CNC permitirá controlar la posición y trayectoria en los ejes **X e Y**, posibilitando la realización de grabados o mecanizados.

Se trata de un proyecto de **alta complejidad**, ya que integra múltiples áreas:
- Electrónica
- Programación
- Diseño 3D
- Mecánica
- Integración de sistemas

---

## ⚙️ Características técnicas

- Microcontrolador: **Arduino UNO**
- Motores: **Motores paso a paso**
- Sensores: **Fines de carrera**
- Sistema mecánico:  
  - A definir entre:  
    - Tornillos sin fin
- Ejes:  
  - X e Y  
  - 2 o 3 motores (según diseño final)

---

## 🧩 Diseño y fabricación

- Diseño en **Autodesk Fusion 360**
- Prototipado mediante impresión 3D
- Material: **PETG**
- Iteración del diseño en base a pruebas reales

---

## 🌐 Software

La interfaz web será desarrollada posteriormente al armado electrónico, para poder adaptar mejor las funcionalidades al sistema real.

Mientras tanto, el equipo de programación irá avanzando en:

## 🔄 Metodología de trabajo

El proyecto se desarrolla de manera **asincrónica**, lo que significa que varias tareas se realizan en paralelo.

Ejemplo:
- Testeo de componentes individuales mientras se diseña
- Desarrollo de software mientras se arma hardware

⚠️ Las fechas pueden modificarse según dificultades del proyecto.
